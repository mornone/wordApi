# Word API 服务图标设计说明

## 设计概念

一个现代化的文档转换图标，体现 Word 到 PDF 的转换功能。

## 设计元素

### 主体图形
- **左侧**：蓝色文档图标（代表 Word）
- **中间**：白色箭头（代表转换）
- **右侧**：红色文档图标（代表 PDF）

### 配色方案
- **主色**：`#2B579A`（Word 蓝）
- **辅色**：`#DC3C2E`（PDF 红）
- **箭头**：`#FFFFFF`（白色）
- **背景**：`#4A90E2`（渐变蓝）

### 尺寸规格
- **推荐尺寸**：256x256 像素
- **最小尺寸**：48x48 像素
- **格式**：ICO（包含多个尺寸：16x16, 32x32, 48x48, 256x256）

## 在线制作步骤

### 方法一：使用 Canva（推荐）

1. 访问 https://www.canva.com/
2. 创建 256x256 像素的设计
3. 添加元素：
   - 圆角矩形背景（蓝色渐变）
   - 左侧文档图标（蓝色）
   - 中间箭头（白色）
   - 右侧文档图标（红色）
4. 导出为 PNG
5. 使用 https://convertio.co/zh/png-ico/ 转换为 ICO

### 方法二：使用 Figma

1. 访问 https://www.figma.com/
2. 创建 256x256 画布
3. 设计图标
4. 导出为 PNG
5. 转换为 ICO

### 方法三：使用 IconArchive（下载现成图标）

1. 访问 https://www.iconarchive.com/
2. 搜索 "document convert" 或 "word pdf"
3. 下载免费图标
4. 确保符合授权要求

## 使用说明

1. 将制作好的图标命名为 `icon.ico`
2. 放在项目根目录：`E:\work\secdriver\WordApiService\MyApp\icon.ico`
3. 重新编译项目：
   ```cmd
   dotnet publish -c Release -r win-x64
   ```
4. 生成的 exe 文件将包含图标

## 临时方案：使用 Windows 默认图标

如果暂时没有图标，可以先注释掉项目文件中的这一行：
```xml
<ApplicationIcon>icon.ico</ApplicationIcon>
```

程序将使用 Windows 默认的应用程序图标。
